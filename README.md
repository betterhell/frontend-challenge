# Catsterest
Небольшой сайт написаный на **React** + **TypeScript**.
<ul>
  <li>Так же использовал Axios для работы с ассинхронным кодом. </li>
  <li>Zustand как стейт-менеджер.</li>
  <li>React-router-dom для навигации.</li>
</ul>
